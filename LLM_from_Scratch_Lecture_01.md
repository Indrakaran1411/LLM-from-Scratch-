## 🕰️ History of Language Models

### 1960s — ELIZA (First Chatbot)
- One of the **first chatbots** ever made by humans
- Very **basic and limited** — could not provide helpful responses
- Failed at simple tasks like recommending resources
- Represented the **state of NLP 50–60 years ago**

### Today — ChatGPT & Modern LLMs
- Ask the same question ELIZA failed at → ChatGPT answers **perfectly**
- Provides: books, online courses, research papers — all **relevant and useful**
- Massive leap in capability over just a few decades

### Key Modern LLMs
| Model | Company | Key Feature |
|-------|---------|-------------|
| ChatGPT | OpenAI | Most popular conversational AI |
| **Llama 3.1** | **Meta (Facebook)** | Highly capable + **Open Source** |

## 🤖 What is Generative AI?

Generative AI is a **broader field** than just LLMs. It includes:

| Type | What it generates |
|------|------------------|
| **Language Models** | Text, conversations, code |
| **Video Models** | AI-generated videos |
| **Audio Models** | AI-generated music/speech |
| **3D Models** | 3D objects and environments |

> **Real-world example:** Vizuara built an AI application for schools — an **MCQ (Multiple Choice Question) generator** — which was impossible to imagine just 4–5 years ago.

## 🔑 Key Terms to Know (Introduced in This Lecture)

| Term | Simple Meaning |
|------|---------------|
| **LLM** | Large Language Model — AI trained on massive text data to understand and generate language |
| **Generative AI** | AI that can create new content (text, video, audio, 3D) |
| **ELIZA** | First chatbot from the 1960s — very primitive |
| **ChatGPT** | Modern powerful LLM by OpenAI |
| **Llama 3.1** | Meta's open-source powerful LLM |
| **Open Source** | Free to use, modify, and study the model |
| **Positional Encoding** | A technique LLMs use to understand word order |
| **Key, Query, Value** | Core components of the Attention mechanism in LLMs |
| **Attention Mechanism** | How LLMs "focus" on relevant parts of text |
